# Search feature

I want to add a vim-like search feature:

- When you press / you go into search mode
- As you type:
  - It highlights the matches (the part of the text, e.g. you search
    for "bob" then it will highlight "bob" in "bobby-the-branch").
  - Jumps to the first result that matches
- Pressing <c-n> and <c-p> jumps between the results
- Pressing <enter> or <esc> goes back to normal mode
